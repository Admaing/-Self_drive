# 本文的理解和认识

随着5G的发展，更大的物联网设备产生大量数据，要满足这些轻量级的设备

- 使用区块链：解决用户和云之间的不信任问题，也使RDIC更高效
- 预防不诚实用户，保护诚实的云，防止不诚实用户骗取补偿
